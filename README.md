# ⚒️ 설치 및 실행

```sh
$ git clone https://github.com/hyunjiyoo/motion-project.git .
$ yarn install
$ yarn start
```

<br>

# 🕹 주요 기능

- 아이템 추가
- 아이템 삭제
- 아이템 리스트업
- 비디오 URL 입력시 정규표현식으로 Video ID만 필터링해서 적합한 `src` 주소 리턴
